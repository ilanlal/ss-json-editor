# About Json Studio

The "Json Studio" is a simple Google Sheets™ add-on that allows users to format and validate JSON data in a Google Sheets™.

## How it works

This add-on uses the [Apps Script](https://developers.google.com/apps-script) platform to provide a custom menu in Google Sheets™ that allows users to format and validate JSON data.
